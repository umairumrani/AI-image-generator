AI Image Generator

This is a web application that generates images using AI based on text prompts.

Features

  • Generate AI images from text descriptions
  • Server-side processing for optimal performance
  • Download generated images
  • Clean, responsive user interface

Important Note

This application requires a server to run. GitHub Pages only hosts static websites and cannot run the server-side code needed for this application.

How to Use This Application

To use this application, you need to:

  1. Clone the repository from GitHub
  2. Install the dependencies with npm install
  3. Start the server with node server.js
  4. Open your browser and navigate to http://localhost:3000
